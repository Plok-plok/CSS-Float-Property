Smart Mirror

A smart mirror is a mirror that displays useful information like time, weather, and news while still functioning as a reflective surface. It's created using a two-way mirror, a display (like a Raspberry Pi screen), and software that manages the content shown.

Wifi Controlled Robot

A WiFi-controlled robot is a robot that can be controlled remotely via WiFi using a smartphone or computer. It typically uses a microcontroller (like Arduino or Raspberry Pi) with a WiFi module to send commands. The project involves combining hardware (motors, sensors) and software to control the robot's movements wirelessly.

3 Wheeled Car Vaccum

A 3-wheeled car vacuum is an autonomous robot designed to clean surfaces. It uses three wheels for mobility and a vacuum system to pick up debris. Controlled by a microcontroller, it often includes sensors for navigation and may feature WiFi connectivity for remote operation. It combines robotics and cleaning technology for automated cleaning.

Drone Med Kit

A Drone Med Kit is a drone that delivers medical supplies like first aid kits or medication to emergency locations. It’s designed for quick deployment in hard-to-reach or disaster areas, improving response times and enhancing healthcare delivery.
